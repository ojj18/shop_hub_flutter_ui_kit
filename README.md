# 🛍️ Shop Hub - Flutter E-Commerce UI Kit

Shop Hub is a modern, clean, and professional Flutter e-commerce UI kit designed to help developers build beautiful online shopping applications faster.

This UI kit contains high-quality screens, reusable widgets, and a clean architecture that follows Flutter best practices.

---

# ✨ Features

* ✅ 40+ Premium Screens
* ✅ Modern & Clean Design
* ✅ Light and Dark Mode Support
* ✅ Fully Responsive Layout
* ✅ Reusable Components
* ✅ Clean Folder Structure
* ✅ Easy to Customize
* ✅ Supports Android & iOS

# 🚀 Getting Started

## 1. Clone the repository

git clone https://github.com/ojj18/shop_hub_flutter_ui_kit.git

## 2. Navigate to project folder

cd shop_hub_flutter_ui_kit

## 3. Install dependencies

flutter pub get

## 4. Run the project

flutter run
